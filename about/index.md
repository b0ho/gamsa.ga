<center>감사농원에서는 최상의 곶감을 제공합니다</center>

## Features

-   Minimal, you can focus on your content
-   Responsive
-   Disqus integration
-   Syntax highlighting
-   Optional post image
-   Social icons
-   Page for sharing projects
-   Optional background image
-   Simple navigation menu
-   MathJax support
